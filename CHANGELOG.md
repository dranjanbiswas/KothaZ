# Changelog

## v1.1.0

- Added KothaZ Android networking library.
- Added Volley-powered request queue.
- Added GET, POST, PUT, PATCH, DELETE helpers.
- Added JSON Object, JSON Array, and String response support.
- Added headers, params, JSON body, timeout, retry, cache, cancel support.
- Added multipart file upload helper.
- Added file download helper.
- Added Gradle/JitPack publishing support.
- Added Java usage guide.
- Added Kotlin usage guide.
- Added Gradle implementation guide.
- Added Windows 10 one-click GitHub upload script.
